# Builds

Сюда помещается сборка проекта в формате **pkg** (один исполняемый файл для macOS).

## Как собрать

Из корня проекта:

```bash
npm run build:pkg
```

Сборка по умолчанию: **macOS Apple Silicon (arm64)**. Для Intel Mac в `scripts/build-pkg.js` замените `node18-macos-arm64` на `node18-macos-x64`.

В папке появятся:
- **BoxParty** — исполняемый файл сервера
- **client/** — собранный веб-клиент (нужен для раздачи браузеру)
- **games/** — папка с игровыми паками (копируется из корня при сборке)

## Как запустить

1. Запустите исполняемый файл (двойной щелчок или из терминала: `./BoxParty`).
2. Откройте в браузере адрес, который выведется в консоль (например `http://192.168.1.5:3000`).
3. Игроки подключаются по тому же адресу с телефонов в той же Wi‑Fi сети.

На macOS при первом запуске может потребоваться: ПКМ по файлу → «Открыть» (из-за Gatekeeper).

---

**При сборке:** предупреждение pkg «Cannot resolve 'mod'» можно игнорировать — это динамический `require` в Express (движок шаблонов). Сервер не использует шаблоны, только статику и API, бинарник работает корректно.
