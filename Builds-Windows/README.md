# Сборка для Windows

Сюда помещается сборка проекта в формате **pkg** (один исполняемый .exe для Windows).

## Как собрать

Из корня проекта (можно с Mac или с Windows, где установлены Node и npm):

```bash
npm run build:pkg:win
```

В папке появятся:
- **BoxParty.exe** — запускаемый файл сервера
- **client/** — веб-клиент
- **games/** — папка с игровыми паками

## Как запустить на Windows

1. Скопируйте папку **Builds-Windows** целиком на Windows-ПК.
2. Дважды нажмите на **BoxParty.exe**.
3. Откройте в браузере адрес из окна (например `http://192.168.1.5:3000`) или перейдите по нему с телефона в той же Wi‑Fi.

Node.js на компьютере ставить не нужно.
